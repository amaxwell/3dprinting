## What is this

At this time, it's a way to distribute 3D printing models of parts for older
test gear, mainly Tektronix and HP. Directory names should be self-explanatory,
and all you really need is to download an STL file and load it into your slicer.

All of my test prints as of June 2026 have been with an Ender 3, and I use
PETG for final production.
